# Credit Card Fraud Detection 💳

## 📌 Project Overview

This project detects fraudulent credit card transactions using machine learning classification techniques.

## 📊 Dataset

* Contains transaction details with anonymized features (V1 to V28)
* Additional features:

  * Time
  * Amount
* Target variable: **Class (0 = Normal, 1 = Fraud)**

## ⚙️ Steps Performed

* Data Preprocessing
* Handling Class Imbalance
* Train-Test Split
* Model Training using Logistic Regression
* Model Evaluation using Precision, Recall, and F1-score

## 🤖 Model Used

* Logistic Regression

## 📈 Result

* Model successfully identifies fraudulent transactions

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
