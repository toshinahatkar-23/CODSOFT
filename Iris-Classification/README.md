# Iris Flower Classification 🌸

## 📌 Project Overview

This project classifies iris flowers into different species based on their measurements using machine learning.

## 📊 Dataset

* Features:

  * Sepal length
  * Sepal width
  * Petal length
  * Petal width
* Target variable: **Species (Setosa, Versicolor, Virginica)**

## ⚙️ Steps Performed

* Data Loading
* Train-Test Split
* Model Training using Logistic Regression
* Model Evaluation

## 🤖 Model Used

* Logistic Regression

## 📈 Result

* Model accurately classifies iris flower species

## 🛠️ Technologies Used

* Python
* Scikit-learn

## 📂 Files

* iris.py → Model code
* IRIS.csv → Dataset
