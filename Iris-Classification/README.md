# Iris Flower Classification 🌸

## 📌 Project Overview

This project classifies iris flowers into different species based on their measurements using machine learning.

## 📊 Dataset

* Features:

  * Sepal length
  * Sepal width
  * Petal length
  * Petal width
* Target variable: **Species (Setosa, Versicolor, Virginica)**

## ⚙️ Steps Performed
* Data Loading
* Train-Test Split
* Model Training using Logistic Regression
* Model Evaluation using Accuracy Score
* Classification Report Analysis
* Confusion Matrix Generation
* Sample Prediction on new data
* Data Visualization (Scatter Plot)
  
## 🤖 Model Used

* Logistic Regression (multiclass classification)

## 📈 Result

Model accurately classifies iris flower species.

Evaluation Metrics:
* Accuracy Score (~100%)
* Classification Report (precision, recall, f1-score)
* Confusion Matrix

Additional Features:
* Sample prediction of new flower data
* Visualization of classification using scatter plot
  
## 🛠️ Technologies Used

* Python
* Scikit-learn

## 📂 Files

* iris.py → Model code
* IRIS.csv → Dataset
