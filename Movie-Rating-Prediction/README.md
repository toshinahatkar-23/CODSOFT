# Movie Rating Prediction 🎬

## 📌 Project Overview

This project predicts the rating of a movie based on features like genre, director, and actors using machine learning techniques.

## 📊 Dataset

* Contains movie details such as:

  * Genre
  * Director
  * Actors
  * Duration
  * Year
* Target variable: **Rating**

## ⚙️ Steps Performed

* Data Cleaning (handled missing values)
* Feature Engineering (converted categorical data using encoding)
* Model Training using Random Forest Regressor
* Model Evaluation

## 🤖 Model Used

* Random Forest Regressor
  
## 📈 Result

*Model predicts movie ratings based on given features.

Evaluation Metrics:
- R2 Score (model performance)
- Mean Squared Error (MSE)

Additional Insights:
- Feature Importance analysis
- Predictions on new movie data
- Visualization of Actual vs Predicted ratings

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

## 📂 Files

* movie.py → Model code
* movies.csv → Dataset
