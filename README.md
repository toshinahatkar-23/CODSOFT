# CODSOFT Data Science Internship 🚀

## 📌 Overview
This repository contains all the tasks completed during the CODSOFT Data Science Internship.  
The projects focus on solving real-world problems using machine learning techniques, data preprocessing, and model evaluation.

---

## 📂 Projects Included

### 🚢 Titanic Survival Prediction
- Predicts whether a passenger survived or not
- Techniques: Logistic Regression, Data Cleaning, Feature Engineering
- Evaluation: Accuracy, Confusion Matrix

---

### 🎬 Movie Rating Prediction
- Predicts movie ratings based on features like genre, director, and actors
- Techniques: Regression Models, Encoding, Feature Engineering
- Evaluation: R2 Score, Mean Squared Error

---

### 🌸 Iris Flower Classification
- Classifies flowers into species (Setosa, Versicolor, Virginica)
- Techniques: Logistic Regression
- Evaluation: Accuracy, Confusion Matrix, Classification Report

---

### 📊 Sales Prediction
- Predicts product sales based on advertising budget
- Techniques: Linear Regression
- Evaluation: R2 Score, MSE

---

### 💳 Credit Card Fraud Detection
- Detects fraudulent transactions in highly imbalanced dataset
- Techniques: Logistic Regression, Feature Scaling, Class Imbalance Handling
- Evaluation: Precision, Recall, F1-score, Confusion Matrix
- Focus: Maximizing recall to detect fraud cases
