# Sales Prediction 📊

## 📌 Project Overview

This project predicts product sales based on advertising expenditure across different platforms.

## 📊 Dataset

* Features:

  * TV advertising budget
  * Radio advertising budget
  * Newspaper advertising budget
* Target variable: **Sales**

## ⚙️ Steps Performed

* Data Loading
* Feature Selection
* Train-Test Split
* Model Training using Linear Regression
* Model Evaluation using R2 Score and Mean Squared Error (MSE)
* Prediction on new data
* Feature Importance Analysis
* Data Visualization (Actual vs Predicted)

## 🤖 Model Used

* Linear Regression (Multiple Linear Regression)

## 📈 Result

* Model predicts sales based on advertising spending.

Evaluation Metrics:
* R2 Score (model performance)
* Mean Squared Error (MSE)

Additional Features:
* Prediction on new advertising data
* Feature importance analysis (impact of TV, Radio, Newspaper)
* Visualization of Actual vs Predicted Sales

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

## 📂 Files

* sales.py → Model code
* advertising.csv → Dataset
